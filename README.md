# Plugins
Repository for HTML component plugins for rich media

## Particle
Particle effect generator for HTML/JS using GSAP

## Profiler
A stepped series of questions with end goal of displaying a best match product. Uses JSON data config to manager questions and possible answers

## SizeManager
control the states of your rich media execution by specifying aspect ratios. Listens for browser/container size changes and either scales content or switches to new state.
